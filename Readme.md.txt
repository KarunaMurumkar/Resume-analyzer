# 🧠 Resume Analyzer

A smart Streamlit-based web application that analyzes a resume against a given job description and provides tailored feedback to help candidates improve their chances of getting shortlisted.

---

## 🚀 Features

- 📄 Upload your resume (PDF)
- 📋 Paste or enter the job description
- ✅ Get similarity score using NLP techniques
- 🔍 Receive actionable tips on how to improve your resume
- ⚡ Lightweight and fast — powered by Python and Streamlit

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🌐
- spaCy / Scikit-learn for NLP
- PyPDF2 for PDF parsing
